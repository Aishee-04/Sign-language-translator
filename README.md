# Sign Language Detection using CNN

This project detects sign language gestures in real time using:

- Python
- OpenCV
- MediaPipe
- CVZone
- TensorFlow/Keras

## Features

- Real-time hand detection
- Hand image preprocessing
- CNN-based gesture classification
- Live prediction with confidence score

## Dataset

Classes:
- Hello
- Thank You
- Yes
- No
- Thumb's Up
- I Love you

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python test.py
```
